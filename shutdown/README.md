# Shutdown linux machine.

    1. For help, just run the script without any argument 
    
```
    └─$ ./shutdown.sh

Usage
 shut [option]

gemstone script for shutdown machine

Options:

        -h 	 Show help
        -i 	 Shutdown with systemctl
        -r 	 Reboot the machine
        -s 	 Power-Off                      
```
